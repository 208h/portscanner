# portscanner
## 使用参数介绍
-H 目标地址(可以是ip和网址，如www.ghtwf01.cn)
-p 目标端口(可以是某一端口或者端口段，如1-10000)
-t 线程(不设置默认线程是20)
## 支持功能
多线程、端口扫描、banner获取、cdn识别
